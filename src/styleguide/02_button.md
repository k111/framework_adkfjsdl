## Button

### Base

`.mod-btn` - ボタンスタイル

    @example
    a.mod-btn(href="#") Link
    button.mod-btn Button

### Size

各種ボタンサイズ

    @example
    a.mod-btn.btn-sm(href="#") Link small
    a.mod-btn.btn-md(href="#") Link midiam
    a.mod-btn.btn-lg(href="#") Link large

### Button Group

`.mod-btn-group` - ボタングループのスタイル

    @example
    <ul class="mod-btn-group">
    <li><a href="#" class="mod-btn">Link</li>
    <li><a href="#" class="mod-btn">Link</li>
    </ul>
